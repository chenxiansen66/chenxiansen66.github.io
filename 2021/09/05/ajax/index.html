Ajax的用途

用来完成网页的局部刷新

Ajax对象属性

  1. readyState 表示状态变化。
    0：请求未初始化，已创建Ajax对象。
    1：open() 方法已经被调用。
    2：send() 方法已经被调用，并且头部和状态已经可获得。
    3：下载中； responseText 属性已经包含部分数据。
    4：下载操作已完成。

  2. status 表示网络请求状态

    200，404…..（HTTP状态码）

  • 创建Ajax对象

  • 绑定onreadystatechange事件用来处理状态变化并获取响应数据

  • 选择请求方式

  • 发送请求

参数传递

  • get请求

    get请求参数和对应值附加在URL后，利用一个?分割，以key=value，多个参数直接用&分割

    所以我们在使用xhr.open()方法时可以将参数以该格式附在请求地址url中

  • post请求

    post请求参数保存在请求体中，所以我们可以将"key=value&key1=value1"字符串放在xhr.send()的函数参数中进行传递。

    但由于ajax发送请求默认Content-Type: text/html;charset=utf-8该解析方式无法正确解析我们所传递的post请求参数，所以我们需要在xhr.send()方法执行前执行前改变其值。

ajax()常见参数

参数名 内容
type 请求方式get/post
url 请求地址
async 默认true异步
data 发送到服务器的数据
dataType 服务器返回的预期数据类型
contentType 请求头
success 请求成功调用该函数
error 请求失败调用此函数
