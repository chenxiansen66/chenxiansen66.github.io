多人协同作业

组织

  • github中点击右上角加号，选择New organization，之后按需求进行设置，并创建组织。
  • 选择 Create a new repository 进行项目仓库的创建（一个组织中可以有多个项目）
  • 在组织内选择 Invite someone 邀请成员

设置版本（打标签）

通过 get log 我们可以查到版本号是一段很长的哈希值，这样的版本号不利于我们使用以及记忆， 而我们看到的一般程序通常以 v1.1 ，v1.12 等作为版本号，这就用到了 打标签设置版本号。

git相关配置

  • 项目配置文件：项目 / .git / config

  • 全局配置文件：用户目录 / .gitconfig

  • 系统配置文件：/ etc / .gitconfig
